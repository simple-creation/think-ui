# ——基于react的轻量级UI组件库
基于react的轻量级组件库，支持按需导入

ui组件：
* Button 按钮组件
* Badge 徽标数组件
* Alert 警告提示组件
* Drawer 抽屉组件
* Message 消息提示框组件
* Notification 消息通知组件
* Modal 对话框组件
* Progress 进度条组件
* Switch 开关组件
* Tag 标签页组件

# 依赖版本
* react: 16.8.6
* react-dom: 6.8.6
* classnames

## 使用

## 安装
``` js
npm install @gethink
```
或者用yarn安装
``` js
yarn add @gethink
```
## 使用
``` jsx
import { 
  Button,
  Empty,
  Progress,
  Tag,
  Switch,
  Drawer,
  Badge,
  Alert
} from 'gethink-ui'
```
